Fat arrow

Fat arrow is also known as Arrow functions.

Syntax

  const functionName=(parameter1,parameter2,....)=>{
    //function body
  }

Example:In inspect

Uses

  • They are concise than regular functions and they also have a shorter syntax.
  • If the function body consists of a single expression, the arrow function automatically returns the result of that expression without needing an explicit return statement.
  • Lexical thisbinding
  • Used for shorter functions
