Array

An array is a special variable that can hold more than one value.

It is an ordered list of values, each of which is called an element and is specified by an index.

An array can hold values of mixed types such as number,string,boolean,null and even other arrays.

Syntax

    
    
        
  1. Array literal:
    2. 
        const arrayName= [value1,value2,....,valueN];
        
  2. Creating an instance of Array directly(using new keyword):
    3. 
        const arrayName=new Array(element0,element1,.....,elementN);
        
  3. Using array constructor(using new keyword):
    4. 
        const arrayName=new Array(arrayLength);
