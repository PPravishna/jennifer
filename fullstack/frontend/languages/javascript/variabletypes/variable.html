Variables

Variables are like container that hold reusable data.

Variables are used to store and manipulate data.

'Var' keyword is a global level variable

They serve as containers that hold values of different types,such as numbers,strings,booleans,objects, or functions.

'var' keyword can be redeclared and reassigned.

let

'let' keyword is a local level variable.

'let' keyword can only be reassigned and cannot be redeclared.

const

'const' keyword can neither be redeclared nor reassigned.
